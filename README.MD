# 2. Methods

- Electric field scales with magnetic drive via an effective coupling and decays dipoleâ€‘like; a leaky integrator surrogate approximates spiking response from field gradients (activating function heuristic).

Safety model

Optimization
- Grid search over B, f, duty, duration with safety gating and efficacy objective (responders fraction). Pareto and â€œbestâ€ safe protocol exported.
